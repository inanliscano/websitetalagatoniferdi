HTML div

The <div> tag is a block-level container that groups together HTML elements. It is often used for layout and styling purposes. The <div> tag has no semantic meaning, but it is widely used for structuring a page.

Here is an example of how to use the <div> tag to group elements together and apply styles.

Code

<!-- Div container example -->
<div class="container">
    <div class="header">Header Content</div>
    <div class="content">Main Content</div>
    <div class="footer">Footer Content</div>
</div>

Output

Header Content
Main Content
Footer Content
