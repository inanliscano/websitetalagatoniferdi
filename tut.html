HTML Lists

In HTML, lists are used to group a set of related items. There are three types of lists:

  • Ordered List (ol)
  • Unordered List (ul)
  • Definition List (dl)

Below is an example showing how to create ordered and unordered lists in HTML.

Code


<-- Unordered List -->
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ul>

<-- Ordered List -->
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Output

Unordered List:

  • Apple
  • Banana
  • Cherry

Ordered List:

  1. First item
  2. Second item
  3. Third item
