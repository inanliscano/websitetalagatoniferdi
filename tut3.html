HTML Comments

HTML comments are used to leave notes in your HTML code. These comments are not displayed in the browser but are visible to developers looking at the source code. HTML comments are written using the following syntax:

<!-- This is a comment -->

Here is an example of how to use HTML comments in your code:

Code

<!-- This is a comment explaining the header section -->
<header>
    <h1>Welcome to the HTML Tutorial</h1>
</header>

<!-- This is a comment explaining the main content section -->
<main>
    <p>Here you can learn about HTML comments!</p>
</main>

Output

Welcome to the HTML Tutorial
Here you can learn about HTML comments!
